# start

```shell
#vue3-admin-plus
git clone  https://github.com/jzfai/vue3-admin-plus.git
pnpm i
pnpm run dev

#micro-service-single
git clone  https://github.com/jzfai/vue3-admin-single.git

run  hugo-admin/HugoApplication
```
